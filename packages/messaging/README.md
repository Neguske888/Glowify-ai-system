# Messaging

Provider-agnostic contracts for messages, envelopes, topics, queues, exchanges, channels, partitions, offsets, producers, consumers, publishers, subscribers, subscriptions, brokers, streams, acknowledgments, delivery, retry, dead-letter handling, ordering, batching, serialization, deserialization, codecs, routing, filtering, priority, scheduling, retention, replay, checkpoints, snapshots, metrics, diagnostics, repositories, providers, registries, and services.

This package defines interfaces, type aliases, enums, metadata models, and reusable error types only. It does not include runtime behavior, broker implementations, networking, persistence, filesystem access, or vendor-specific assumptions.
