# Integration Runtime

Provider-agnostic contracts for integrations, connectors, connector instances, capabilities, configuration, connections, credential references, secret references, installations, synchronization, webhook handling, polling, scheduling, execution, retry, failures, health, diagnostics, metrics, repositories, registries, and services.

This package defines interfaces, enums, type aliases, metadata models, and reusable error types only. It does not include runtime implementation, HTTP, SDKs, persistence, queues, filesystem access, or business logic.
