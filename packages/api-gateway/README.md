# API Gateway

Provider-agnostic contracts for gateway metadata, requests, responses, routing, middleware, serialization, validation, authentication, authorization, rate limiting, throttling, quotas, caching, compression, CORS, versioning, negotiation, streaming, WebSocket, GraphQL, gRPC, REST, SSE, health, diagnostics, metrics, repositories, providers, registries, and services.

This package defines interfaces, type aliases, enums, metadata models, and reusable contracts only. It does not include runtime behavior, HTTP implementation, networking, persistence, filesystem access, SDK imports, or vendor-specific assumptions.
