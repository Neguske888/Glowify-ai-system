# Runtime

Provider-agnostic contracts for runtime metadata, kernel metadata, bootstrap, startup, shutdown, health, readiness, liveness, dependency graphs, resources, workers, processes, execution, diagnostics, metrics, telemetry, configuration, registries, repositories, providers, and services.

This package defines interfaces, type aliases, enums, metadata models, and reusable error types only. It does not include runtime behavior, filesystem access, networking, persistence, SDK imports, or assumptions about any execution environment.
